

Cтруктура проекта

src/
│── ── ──  app
│
│── ── ── pages (Страницы приложения)
│
│── ── ── entities (Хранение сущностей)
│
│── ── ── shared (Общие модули)
│               ├────── api (Логика взаимодействия с API)
│               ├────── const (константы)
│               ├────── locales (локализация)
│               ├────── redux (вся логика redux)
│               ├────── UI (UI - компоненты)
│
│── ── ── features   Фичи (основная бизнес-логика)
│
│
│── ── ── widgets (группы компонентов)
│
│
│── ── ── assets (иконки, изображения, шрифты, прочее)
